README.TXT

This code is for the NAD - Name and Address system and was an independent tool
for the entry and maintenance of such data and the production of address labels.

It is written in Basic using the CBASIC compiler in a CPM platform.
The copyright and usage notices as used within the ACAS system, applies to this
code base.

If you decide to use a Basic compiler such as the Free Basic Compiler, you will
need to make a number of changes to this code to match with what the FBC
accepts.

I cannot find the manual in source or printed form but hopefully that should not
be a problem as reading the Basic code should be enough.

On a very quick glance though the code I am not sure that all code uses the
Postcode field instead of the original zipcode so it needs to be tested for,
(I could be wrong).


Have fun.
